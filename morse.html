morsepearls

2022

Morsepearls Secret Code Language, born from an information design lecture, teaches the creation of two-way information systems. Using 45 symbols inspired by Morse code, it introduces a playful, visually engaging secret code system for abstract message communication. The bead-inspired design simplifies reading, and clear starting indicators facilitate decoding for constructing meaningful sentences.
